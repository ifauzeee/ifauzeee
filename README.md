## ï¿½ Quick Start Deployment Guides

<details>
<summary><b>ğŸ“¦ Deploy a Next.js Project (e.g., Zee-Index, WhisperBin)</b></summary>
<br/>

### Prerequisites
- Node.js 18+ installed
- pnpm, npm, or yarn package manager
- Git installed

### Step 1: Clone & Install
```bash
# Clone the repository
git clone https://github.com/ifauzeee/<project-name>.git
cd <project-name>

# Install dependencies
pnpm install
# or: npm install
```

### Step 2: Configure Environment
```bash
# Copy environment template
cp .env.example .env

# Edit .env with your credentials
# Each project includes documentation for required variables
```

### Step 3: Run Development Server
```bash
pnpm dev
# Access at http://localhost:3000
```

### Step 4: Deploy to Vercel (Recommended)
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Follow prompts to link and deploy
```

**Alternative: One-Click Deploy**

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

</details>

<details>
<summary><b>ğŸ³ Deploy a Python Bot (e.g., Zee-Mirror, Zee-UBot)</b></summary>
<br/>

### Prerequisites
- Docker & Docker Compose installed
- Telegram API credentials from [my.telegram.org](https://my.telegram.org)
- Python 3.10+ (for session generation)

### Step 1: Clone Repository
```bash
git clone https://github.com/ifauzeee/<bot-name>.git
cd <bot-name>
```

### Step 2: Generate Session String
```bash
# Install dependencies locally
pip install pyrogram tgcrypto

# Generate your session string
python gen_session.py
# Follow prompts to login
```

### Step 3: Configure Environment
```bash
# Copy example environment file
cp .env.example .env

# Edit with your credentials:
# - API_ID
# - API_HASH
# - SESSION_STRING
# - BOT_TOKEN (if applicable)
```

### Step 4: Deploy with Docker
```bash
# Build and run in detached mode
docker compose up -d --build

# View logs
docker compose logs -f

# Stop the bot
docker compose down
```

### Step 5: Local Python (Alternative)
```bash
# Install dependencies
pip install -r requirements.txt

# Run
python main.py
```

</details>

```bash
# Run with Docker Compose

```
